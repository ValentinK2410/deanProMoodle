## Создание профильного заголовка

### HTML структура

```php
function render_student_profile_header($user) {
    global $OUTPUT;
    
    $photo = $OUTPUT->user_picture($user, ['size' => 120]);
    $fullname = fullname($user);
    $email = $user->email;
    
    $html = html_writer::start_div('student-profile-header-main');
    
    // Фото
    $html .= html_writer::start_div('student-profile-photo-main');
    $html .= html_writer::link(
        new moodle_url('/user/profile.php', ['id' => $user->id]),
        $photo,
        ['class' => 'd-inline-block aabtn', 'target' => '_blank']
    );
    $html .= html_writer::end_div();
    
    // Информация
    $html .= html_writer::start_div('student-profile-info-main');
    $html .= html_writer::tag('h1', 
        html_writer::link(
            new moodle_url('/user/profile.php', ['id' => $user->id]),
            $fullname,
            ['class' => 'student-profile-name-link', 'target' => '_blank']
        ),
        ['class' => 'student-profile-name-main']
    );
    $html .= html_writer::start_div('student-profile-email-main');
    $html .= html_writer::link('mailto:' . $email, $email);
    $html .= html_writer::end_div();
    $html .= html_writer::end_div();
    
    $html .= html_writer::end_div();
    
    return $html;
}
```

### PHP код для пагинации

```php
function render_pagination($currentpage, $totalpages, $baseurl) {
    $html = html_writer::start_div('pagination-wrapper');
    
    // Кнопка "Назад"
    if ($currentpage > 0) {
        $prevurl = new moodle_url($baseurl, ['page' => $currentpage - 1]);
        $html .= html_writer::link($prevurl, '◀ Назад', ['class' => 'pagination-button']);
    } else {
        $html .= html_writer::tag('span', '◀ Назад', ['class' => 'pagination-button', 'style' => 'opacity: 0.5; cursor: not-allowed;']);
    }
    
    // Информация о странице
    $html .= html_writer::tag('span', 
        "Страница " . ($currentpage + 1) . " из " . $totalpages . " записей",
        ['class' => 'pagination-info']
    );
    
    // Кнопка "Вперед"
    if ($currentpage < $totalpages - 1) {
        $nexturl = new moodle_url($baseurl, ['page' => $currentpage + 1]);
        $html .= html_writer::link($nexturl, 'Вперёд ▶', ['class' => 'pagination-button']);
    } else {
        $html .= html_writer::tag('span', 'Вперёд ▶', ['class' => 'pagination-button', 'style' => 'opacity: 0.5; cursor: not-allowed;']);
    }
    
    $html .= html_writer::end_div();
    
    return $html;
}
```

### JavaScript для модального окна

```javascript
function openModal(url, title) {
    var overlay = document.getElementById('deanpromoodle-modal-overlay');
    var iframe = document.getElementById('deanpromoodle-modal-iframe');
    var modalTitle = document.getElementById('deanpromoodle-modal-title');
    
    if (overlay && iframe && modalTitle) {
        modalTitle.textContent = title;
        iframe.src = url;
        overlay.classList.add('active');
        document.body.style.overflow = 'hidden';
    }
}

function closeModal() {
    var overlay = document.getElementById('deanpromoodle-modal-overlay');
    var iframe = document.getElementById('deanpromoodle-modal-iframe');
    
    if (overlay) {
        overlay.classList.remove('active');
        document.body.style.overflow = '';
    }
    if (iframe) {
        iframe.src = '';
    }
}

// Закрытие по Escape
document.addEventListener('keydown', function(e) {
    if (e.key === 'Escape') {
        closeModal();
    }
});
```
